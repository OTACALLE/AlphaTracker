# Alpha Tracker Vision

Alpha Tracker é uma plataforma de vigilância científica para biologia estrutural.

Objetivo:

Detectar automaticamente proteínas recém-publicadas cuja sequência completa de aminoácidos seja conhecida e para as quais não exista evidência pública de modelo estrutural ou publicação estrutural ou que alguem que tenha rodado no AlphaFold publicou os resultados

O sistema deve funcionar continuamente.

Ele deve monitorar dezenas de bases científicas.

Ele deve calcular um score científico.

Ele deve alertar automaticamente o pesquisador.

Ele deve servir como ferramenta de prospecção científica.
